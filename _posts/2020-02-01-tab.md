To insert a tab character in `bash` you can do `CTRL-V` followed by `TAB` or only write `\t`. Here is an example:

```bash
[murphy@eva02]$ head -n 1 datos.tsv | awk -F "        " ' {print $1} '
pozo
[murphy@eva02]$ head -n 1 datos.tsv | awk -F "\t" ' {print $1} '
pozo
```
