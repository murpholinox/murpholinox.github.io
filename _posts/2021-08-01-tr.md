tag: command line
---

```bash
cat lolo.tsv | tr '\t' ' '
lolo lola lulu lilo 
xoxo xoxa xuxu xixo
cat lolo.tsv | tr '\t' ', '
lolo,lola,lulu,lilo,
xoxo,xoxa,xuxu,xixo
cat lolo.tsv | tr '\t' ', ' > nuevo.csv
cat nuevo.csv | tr ',' '\t'
lolo	lola	lulu	lilo	
xoxo	xoxa	xuxu	xixo
```

Very simple, right?
